SOFÍA v1.0 · 2026
Sistema Offline de Funciones con Inteligencia Artificial
Asistente de voz personal en español, 100% offline, para Windows. Sin suscripciones, sin Internet, sin privacidad comprometida. STT, TTS e IA conversacional corren completamente en tu equipo.
🔒 100% Offline 🎙️ Activación por voz ✓ Sin API keys 🇪🇸 Español nativo 🖥️ Control de escritorio ⚡ Open source
¿Qué puede hacer?
  • 🎙️
    Activación por voz — Di "Sofía" y espera el tono. Detección difusa (Levenshtein) para variaciones de pronunciación.
  • 🌤️
    Clima en tiempo real — "Sofía, ¿qué clima hay en Bogotá?" vía Open-Meteo, sin API key.
  • 📅
    Agenda completa — Eventos, alarmas y tareas. Compartida con NebulaNotes (JSON compatible).
  • 🖥️
    Control de escritorio — Abrir, cerrar, minimizar/maximizar apps. Resolver referencias implícitas: "ciérrala", "minimízala".
  • 🧠
    IA conversacional — Qwen3-8B local vía llama.cpp. Responde preguntas libres en español, sin conexión.
  • 📝
    Notas rápidas — "Sofía, crea una nota: reunión con el cliente el viernes."
  • 🌐
    Búsqueda web y YouTube — Lanza búsquedas directamente desde la voz.
  • ⏰
    Rutina matutina — "Buenos días, Sofía" despliega saludo + clima + agenda del día.
Pipeline de voz
1
Silero-VAD — Filtra silencio en tiempo real (CPU, siempre activo)
↓
2
Whisper "tiny" — Detecta la wake-word "Sofía" (match difuso)
↓
3
Reproduce "Dime" (audio pre-renderizado, sin GPU, instantáneo)
↓ ← carga perezosa en paralelo
4
Whisper "base" — Transcribe el comando completo (más preciso)
↓
5
Router — Primer keyword-match gana; fallback al LLM
↓
6
TTS — pyttsx3 (offline rápido) o Qwen3-TTS (GPU, natural)
El tiempo de carga de los modelos pesados queda oculto en el tiempo que el usuario tarda en pronunciar su comando.

Arquitectura: 5 principios
Router por keywords
Sin NLU. Latencia mínima. Depurable. Determinista.
LLM solo fallback
El 95% de comandos los resuelven skills sin tocar el LLM.
Lazy loading
Modelos pesados cargan mientras el usuario habla.
JSON plano
Sin base de datos. Un usuario, volumen trivial.
Doble Whisper
tiny (wake-word) + base (comando). Costo mínimo en espera.
Privacidad by design
Ningún dato personal cruza la red. Nunca.
Extensibilidad: 2 pasos
Integra SOFÍA con cualquier sistema sin tocar el núcleo:
# 1. skills/mi_skill.py KEYWORDS = ["mi comando", "mi tarea"] def manejar(texto) -> str: return "Hecho." # 2. main.py router.registrar("mi_skill", mi_skill.KEYWORDS, mi_skill.manejar)
→ Riesgo de regresión prácticamente nulo. El núcleo ya probado no cambia.
